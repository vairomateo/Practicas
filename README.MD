Querandí Mates

Este proyecto es mi entrega final del curso de desarrollo web.  
La idea fue crear un sitio completo para una marca llamada Querandí Mates, una tienda dedicada a productos relacionados con el mate: mates, bombillas, yerbas y más.

El sitio está hecho desde cero aplicando todo lo aprendido: estructura HTML, estilos con SASS, uso de Bootstrap, SEO, animaciones, y diseño responsive.

## Link al sitio

https://vairomateo.github.io/Entrega-Final/

## Tecnologías usadas

- HTML5
- CSS3 / SASS 
- Bootstrap 5 
- Flexbox y Grid  
- Git / GitHub Pages
- SEO

##  Qué tiene el sitio

- Diseño responsivo que se adapta bien a distintas pantallas  
- Secciones con contenido realista y coherente (Inicio, Productos, Nosotros, Dudas, Contacto)  
- Formularios con validación  
- Animaciones y efectos hover  
- Footer con links e íconos de redes sociales  
- Favicon, meta descriptions y keywords para SEO  

## Sobre el diseño

Busqué un estilo limpio y moderno, que mantenga una estética artesanal y argentina.  
La navegación es clara, los textos son legibles y las imágenes acompañan bien al contenido.  
Todo el sitio está hecho pensando en la experiencia del usuario, el rendimiento y el SEO.
